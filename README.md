## Requirements

- Node >= 20
- typescript
- pnpm
- postgresql

## Development

- **(ℹ️ only for first setup)** Create manually your DB via psql or TablePlus
- Run `./scripts/reset-db` to setup database and seed
- Run `./scripts/dev` to start development server

## Tools

- ESlint
- Prettier
